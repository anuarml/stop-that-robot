# Stop that robot

Application to simulate a robot

# Approximate time spent on this project

16 hours

# Instructions

To start the application run `npm run start`
To execute the application tests run `npm run test`
To build the applicatoin for productoin run `npm run build`

# Notes

This application can be used with keyboard, just tab until you reach the section where all the buttons are and then use the arrows to move the Robot after turning it on.
